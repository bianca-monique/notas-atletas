# 🤸 Notas de Atletas

Aplicação em JavaScript que calcula a média das notas de atletas de ginástica artística.

## 📋 Sobre o projeto

Em uma competição com 5 jurados, cada um avalia um critério diferente.
A média é calculada com base nas **3 notas do meio**, desconsiderando a maior e a menor.

## 🚀 Como executar

1. Faça o download do arquivo `notas-atletas.js`
2. Abra o terminal na pasta do arquivo
3. Execute o comando: `node notas-atletas.js`

## 💻 Tecnologias

- JavaScript
